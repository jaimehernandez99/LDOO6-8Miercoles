UTP

UTP (Micro Transport Protocolo) es un protocolo libre multiplataforma que fue diseñado para ser utilizado en las conexiones P2P en la red bittorrent, está implementado sobre el protocolo UDP, como alternativa a TCP para la transferencia de datos. Este protocolo sirve para prevenir la latencia, pero aprovechando el ancho de banda cuando la latencia no es excesiva.
